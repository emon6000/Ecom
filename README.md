# 🪄 MAGIC SHOP

A modern, fully responsive e-commerce web application designed to bring a touch of enchantment to online shopping. Built with the MERN stack, this project features a sleek dark-slate and neon-green aesthetic, dynamic product filtering, and an optimized image delivery system.

## ✨ Features

* **Fully Responsive UI:** Seamless shopping experience across desktop, tablet, and mobile devices using CSS Grid and Flexbox.
* **Dynamic Search & Sorting:** Real-time product filtering by name and category, with price sorting capabilities.
* **Interactive Hero Carousel:** Auto-playing promotional banner with manual navigation controls.
* **Cart Management:** Intuitive state management for adding items and tracking cart quantities.
* **Optimized Image Hosting:** Integrated with Cloudinary for automatic on-the-fly image compression and modern WebP formatting.
* **Modern Aesthetic:** Custom styled with a premium dark theme and vibrant neon accents.

## 🛠️ Tech Stack

**Frontend:**
* React.js
* React Router DOM
* Context API (State Management)
* Custom CSS3 (CSS Grid & Flexbox)

**Backend:**
* Node.js
* Express.js
* Axios

**Database & Storage:**
* MongoDB
* Cloudinary (Image CDN with auto-compression `f_auto,q_auto,c_limit,w_1200`)
